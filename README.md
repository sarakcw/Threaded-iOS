# Threaded Community App

MonCraft is an iOS app designed for knitters, crocheters, and embroiderers to connect, share, and track their creative projects.

## Features

- **Real-time Community Interaction:** Integrated Firebase Firestore for authentication and real-time updates on events and user activity.
- **Offline Project Tracking:** Used Core Data to store projects and progress locally, allowing users to continue tracking even without internet.
- **Pattern Recommendations:** Embedded Ravelry API to provide free pattern suggestions tailored to user interests.
- **Event Discovery:** Implemented MapKit and CoreLocation to help users find local crafting events based on proximity.
- **Intuitive UI:** Designed a user-friendly and accessible interface for smooth navigation and community engagement.

## Tech Stack

- iOS (Swift, UIKit)
- Firebase Firestore
- Core Data
- Ravelry API
- MapKit & CoreLocation

## Overview

Threaded allows users to:
- Connect with other crafters in their area.
- Share progress on knitting, crochet, and embroidery projects.
- Discover free patterns and crafting events.
- Track personal projects both online and offline.
